# <ins>While statement</ins>

## Penggunaan while
```
while (true)
{
    // Block code
}
```
contoh:
```
int i = 0;
while (i < 5) 
{
  Console.WriteLine(i);
  i++;
}
```

## Penggunaan do/while
mengeksekusi code block, baru mengcek kondisi
```
do 
{
  // Block code
}
while (condition);
```
contoh:
```
int i = 1;
do
{
    Console.WriteLine(i);
    i++;
}
while (i < 5);
```
