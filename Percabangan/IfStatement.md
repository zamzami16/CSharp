# <ins>IF Statement</ins>

## Penggunaan
```
if (true)
{
    // Block code here
}
else if (true)
{
    // Block code here
}
else
{
    // Block code here
}
```
contoh:
```
if (num == 1)
{
    Console.WriteLine("1 betul");
}
else if (num == 2)
{
    Console.WriteLine("2 Betul");
}
else
{
    Console.WriteLine("lainnya");
}
```
